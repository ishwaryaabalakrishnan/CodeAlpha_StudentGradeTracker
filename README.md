# Student Grade Tracker

## Project Description
Student Grade Tracker is a Java application developed as part of the CodeAlpha Java Programming Internship. It helps users manage student records, calculate average marks, identify the highest and lowest scores, and display a summary report.

## Features
- Add student names and marks
- Calculate average marks
- Find highest and lowest marks
- Display student summary report
- Uses Java OOP concepts
- Uses ArrayList for storing data

## Technologies Used
- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## How to Run
1. Open the project in IntelliJ IDEA.
2. Run `StudentGradeTracker.java`.
3. Enter student details.
4. View the generated report.
